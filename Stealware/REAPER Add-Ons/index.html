Mason's Workshop Scripts for REAPER

These are various utilities I have made over the years. Some simple, some . There are a few caveats you should know before you dig in:

  1. I am on WINDOWS, and I have only ever used Mac under duress. Therefore, I these will all work on Mac, particularly the more oddball ones involving text-to-speech, FFMPEG, transcription, etc. However, MOST of these should work just fine!
  2. You must have the SWS/S&M Extension for REAPER installed. Not just for my scripts, but because it offers SO much functionality!
How to Add Scripts to REAPER
  • Download the .lua file you want to have and save it somewhere memorable.
  • Launch REAPER, and open the Actions menu. (Hotkey: "Shift + ?")
  • Select "New action..." and then "Load ReaScript..."
  • Use the file browser to select the .lua file you downloaded earlier!
  • Profit.
